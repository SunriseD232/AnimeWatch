Автоматический iframe-плеер
Код для установки
Для установки достаточно разместить в том месте, где вы хотите вывести плеер этот код:
<iframe src="https://tv-2-kinoserial.net/embed_auto/ID/?token=ТОКЕН" width="100%" height="400px" frameborder="0" allowfullscreen allow="autoplay *; fullscreen *"></iframe>
где:
ID - ID кинопоиска/IMDB ID/для TMDB перед id укажите tmdb (например tmdb575089),
100% - ширина iframe в пикселях или процентах,
400px - высота в пикселях или процентах,
ТОКЕН - ваш токен.
Как установить в DLE
1. Откройте шаблон полной новости (fullstory.tpl).
2. В том месте, где вы хотите вывести плеер добавьте этот код:
[xfgiven_kpid]<iframe src="https://tv-2-kinoserial.net/embed_auto/[xfvalue_kpid]/?token=ТОКЕН" width="100%" height="400px" frameborder="0" allowfullscreen allow="autoplay *; fullscreen *"></iframe>[/xfgiven_kpid]
где:
kpid - название вашего дополнительного поля с ID кинопоиска,
100% - ширина iframe в пикселях или процентах,
400px - высота в пикселях или процентах,
ТОКЕН - ваш токен.
Поддерживаемые параметры
default_audio_id
Выбирает звуковую дорожку. В качестве аргумента передается ID звуковой дорожки.
https://tv-2-kinoserial.net/embed/249/?token=ТОКЕН&default_audio_id=11
video
Запуск определенного видео из сезона/сериала. В качестве аргумента передается: vСЕРИЯ - для сезона, sСЕЗОНvСЕРИЯ - для сериала.
https://tv-2-kinoserial.net/embed_season/77/?token=ТОКЕН&video=v11 - 11 серия сезона.
https://tv-2-kinoserial.net/embed_season/4400/?token=ТОКЕН&video=s7v8 - 7 сезон 8 серия сериала.
only
Работает только совместно с параметром video. Поддерживается страницами embed_auto, embed_serial, embed_season. Позволяет вместо сериала/сезона вывести плеер с конкретным сезоном/видео. В качестве аргумента принимает значение true.
https://tv-2-kinoserial.net/embed_serial/686/?token=ТОКЕН&video=s12v0&only=true - выведет плеер только с 12 сезоном, так как v0.
https://tv-2-kinoserial.net/embed_serial/686/?token=ТОКЕН&video=s12v3&only=true - выведет плеер только с 3 серией 12 сезона.
https://tv-2-kinoserial.netembed_season/1795/?token=ТОКЕН&video=v3&only=true - выведет плеер только с 3 серией сезона.
start
Запуск видео с заданной секунды. В качестве аргумента передается секунда видео.
https://tv-2-kinoserial.net/embed/249/?token=ТОКЕН&start=360
Комбинирование
Все параметры можно комбинировать в любой очередности. Сочетание определяется доступными дорожками и субтитрами.
https://tv-2-kinoserial.net/embed/249/?token=ТОКЕН&default_audio_id=11&start=360

JS скрипт
Для установки достаточно разместить JS скрипт.
Разместите этот код перед закрывающим тегом </head>:
<script src="https://tv-2-kinoserial.net/static/js/addiframe.js"></script>:

Разместите этот код в том месте, где требуется вывести плеер:
<div id='iframe'></div>

Разместите этот код перед закрывающим тегом </body>:
<script>addtoiframe('iframe', 'KP_ID', 'ШИРИНА', 'ВЫСОТА', 'ТОКЕН');</script>
где: KP_ID - ID кинопоиска, ШИРИНА - ширина iframe в пикселях или процентах, ВЫСОТА - высота в пикслеях или процентах, ТОКЕН - ваш токен.

Пример использования:
<script>addtoiframe('iframe', '1238599', '100%', '400px', 'TOKEN');</script>





API v2
Основная информация
API-токен можно получить на странице редактирования профиля.
URL API https://api.videoseed.tv/apiv2.php

Обязательные параметры:
token - Ваш токен
list - список контента. Может принимать следующие значения:
movie - фильмы и мультфильмы
serial - сериалы и мультсериалы
category - список жанров
country - список стран
voice - список озвучек
item - вся информация об одной единице контента. Может принимать следующие значения:
movie - фильмы и мультфильмы
serial - сериалы и мультсериалы
search - выбирает одну единицу контента по kp, imdb, tmdb вне зависимости от ее типа.
Дополнительные параметры
from - номер страницы, начиная с 1.
items - количество элементов на один запрос. По-умолчанию: 12.
sort_by - параметры сортировки. Может принимать следующие значения:
post_date asc - дата публикации по возрастанию.
post_date desc - дата публикации по убыванию.
post_date asc - дата последнего обновления контента по возрастанию.
post_date desc - дата последнего обновления контента по убыванию.
kp asc - KP ID по возрастанию.
kp desc - KP ID по убыванию.
title asc - название по возрастанию.
title desc - название по убыванию.

Фильтры
id - Внутренний ID.
kp - Kinopoisk ID.
imdb - IMDB ID.
tmdb - TMDB ID.
q - поисковый запрос. Ищет по русскому и английскому названию.
release_year_from - минимальный год выхода.
release_year_to - максимальный год выхода.
categories - ID жанра или страны, можно указать несколько через запятую.
Пример запроса
https://api.videoseed.tv/apiv2.php?item=serial&token=ВАШ_ТОКЕН&sort_by=kp%20asc&q=ТЕКСТ - URL-запроса
